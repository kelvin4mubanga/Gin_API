# Gin API Collection

This repository contains a diverse set of backend RESTful APIs built with the [Gin Web Framework](https://github.com/gin-gonic/gin) in Go. Each API targets a specific domain, demonstrating practical, real-world backend solutions with clean architecture and RESTful design.

---

## APIs Included

- **Blog API** — CRUD operations for blogging platform  
- **Student Management API** — Manage students, courses, and enrollments  
- **Cloud Storage API** — File upload, download, and management  
- **Subscription API** — User subscriptions and payment handling  
- **Dating API** — User profiles, matching, and messaging  
- **Todo API** — Task management system  
- **Digital Wallet API** — Wallet balances, transfers, and transactions  
- **Travel API** — Browse and explore travel destinations  
- **E-commerce API** — Products, carts, and orders management  
- **Travel Booking API** — Book trips and travel packages  
- **Job Listing API** — Post and apply for jobs  
- **Vehicle Reservation API** — Reserve and manage vehicle bookings  
- **Library Management API** — Books, users, loans management  
- **Micro Lending API** — Loan application and approval system  
- **Online Auction API** — Create and bid on auctions  

---

## Technology Stack

- **Language:** Go (Golang)  
- **Framework:** Gin Web Framework  
- **Database:** SQLite / (using GORM ORM)  
- **Authentication:** JWT (JSON Web Tokens)(not yet implemented)
- **Version Control:** Git & GitHub
